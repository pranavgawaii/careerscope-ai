## üéØ Features

### ‚úÖ Resume Analyzer (Active)
Stop guessing keywords. Our ATS-style scoring engine mimics recruiter logic.
- **ATS Scoring:** Get a concrete match score (0-100).
- **Keyword Gap Detection:** Identify missing skills crucial for your target role.
- **Bullet Point Optimizer:** Specific rewrite suggestions to quantify your impact (e.g., turning "Worked on API" into "Scaled REST API to 10k users").

### üéôÔ∏è Interview Evaluator (Coming Soon)
Practice makes perfect.
- **Mock Interviews:** Answer frequent technical and behavioral questions.
- **Instant Feedback:** Get graded on clarity, structure (STAR method), and relevance.

### üíª GitHub Project Analyzer (Coming Soon)
For developers, code is the new resume.
- **Repo Audit:** Automated checks for README quality, code structure, and documentation.
- **Recruiter Readiness:** Ensures your pinned projects showcase your best engineering skills.

## üöÄ Getting Started

To run this project locally, follows these simple steps:

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Python 3.x (optional, for local server) or any static file server

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/pranavgawaii/careerscope-ai.git
   cd careerscope-ai
   ```

2. **Start a local server**
   Since this is a static site, you can open `index.html` directly, but for the best experience (avoiding CORS issues with some assets), run a simple server:
   
   Using Python:
   ```bash
   python3 -m http.server 8080
   ```

3. **View the application**
   Open your browser and navigate to:
   ```
   http://localhost:8080
   ```
